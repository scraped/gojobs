# Important cookies & headers 

Must be included in every request—é 

| Type   | Necessary | Name               | Value                      |
| ------ | --------- | ------------------ | -------------------------- |
| Header | `y`       | `x-requested-with` | `XMLHttpRequest`           |
| Header | `y`       | `x-amc`            | `true`                     |
| Header | `n`       | `x-lang`           | `en-US`, `ru-RU` and so on |
